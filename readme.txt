Enables Single Sign-On via SSL for BizLibrary resources.

== Description ==

This plugin provides seamless access to [BizLibrary](http://www.bizlibrary.com/) catalog of over 6,000 courses. Request a Single-Sign On authentication service with your BizLibrary account, fill out the Access Key, Secret, and Site code and you will be able to link to your BizLibrary control panel. 

This plugin also comes with availability to import a BizLibrary course record file. Fill in the BizLibrary SFTP login info (provided with your BizLibrary account and your user's courses will be available within their user meta data.

== Installation ==

1. Upload `bizlibrary` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
